Generatore di canzoni


  • Puoi caricare un file testuale con un elenco di canzoni (nome, ID, link).
  • Un pulsante permette di estrarre una canzone casuale dal file caricato.
  • Per ogni estrazione verranno mostrati: nome, ID e link della canzone.
  • Non possono uscire due volte di seguito le stesse canzoni.
  • Quando tutte le canzoni saranno state estratte, apparirÃ  il messaggio: "Canzoni terminate."
